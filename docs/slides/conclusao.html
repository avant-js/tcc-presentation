CONCLUSÃO


Modelos clássicos não suportam a totalidade das representações necessárias para aplicações web modernas


Foi possível desenvolver uma aplicação que cobre os requisitos de micro serviços e teve grande aceitação dos usuários

Usuários favoráveis ao método de modelagem proposto e à geração de código a partir do modelo


Os usuários buscam maneiras de ter uma visão geral das aplicações, mas que efetivamente ajudem no desenvolvimento, tornando-o mais rápido e melhorando a qualidade das entregas
